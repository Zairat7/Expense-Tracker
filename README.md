# Expense-Tracker
GUI-приложение «Expense Tracker» для отслеживания личных расходов с фильтрацией, подсчётом суммы за период, JSON и Git.
# Expense Tracker
**Автор:** [Бегидова Заира]

## Описание
Программа для мониторинга личных финансов. Данные хранятся локально в формате JSON.

## Новые функции
- **Фильтрация по категориям:** возможность смотреть расходы только на "Еду" или "Транспорт".
- **Временные периоды:** установка диапазона дат "С" и "По" для анализа трат за месяц/год.
- **Динамический подсчет:** сумма "Итого" пересчитывается автоматически при применении фильтров.

## Инструкция по использованию
1. **Добавление:** Заполните поля сверху и нажмите "Добавить". Данные тут же сохранятся в файл `expenses.json`.
2. **Фильтрация:** В блоке фильтров выберите нужную категорию или измените даты. Нажмите "Применить фильтр".
3. **Проверка данных:** Если ввести отрицательное число или текст в поле суммы, программа выдаст предупреждение.

## Примеры тестов
1. Введите дату `2024-05-01`, сумму `500`. 
2. Введите дату `2024-06-01`, сумму `1000`.
3. Установите фильтр "По" на `2024-05-31`. В таблице останется только первая запись, сумма "Итого" станет `500`.
