# Expense-Tracker
GUI-приложение «Expense Tracker» для отслеживания личных расходов с фильтрацией, подсчётом суммы за период, JSON и Git.
# Expense Tracker
**Автор:** [Бегидова Заира]

## Описание
Приложение для учета личных финансов. Позволяет добавлять расходы, сохранять их в JSON и просматривать общую сумму.

## Как использовать
1. Установите Python 3.x.
2. Запустите файл: `python main.py`.
3. Введите сумму, выберите категорию и дату.
4. Нажмите «Добавить расход».

## Тест-кейс
- Сумма: `1500`, Категория: `Еда`, Дата: `2024-05-20` -> Успешно добавлено.
- Сумма: `-50` -> Ошибка валидации.
